# Mês 1 — Fundamentos do Computador

Semanas 1 a 4. Aqui você aprende o básico: mexer no computador, organizar arquivos e usar a internet.

**Ao final deste mês você vai saber:**
- Navegar pelo Windows e personalizar o computador
- Criar, renomear, copiar e organizar arquivos e pastas
- Pesquisar na internet, salvar favoritos e baixar arquivos
- Compactar e extrair arquivos ZIP

## Semana 1 — Conhecendo o PC

> Objetivo: saber navegar pelo Windows, personalizar o computador e criar sua estrutura de pastas
> Tempo estimado: 2–3 horas

### Missão 1 — Explorar a Área de Trabalho

A área de trabalho é a tela principal do Windows. Vamos conhecer os elementos dela.

1. Olhe para a barra de tarefas (a barra na parte inferior da tela). Identifique os ícones fixados
2. Clique no botão **Iniciar** (ícone do Windows na barra de tarefas) e explore o menu
3. Abra as **Configurações**: clique em Iniciar > ícone de engrenagem, ou use `Win + I`
4. Abra o **Explorador de Arquivos** usando `Win + E`
5. Use `Alt + Tab` para alternar entre as janelas que você abriu

> **Dica:** `Win + D` mostra a área de trabalho rapidamente. Pressione de novo para voltar.

### Missão 2 — Personalizar o Computador

Deixe o computador com a sua cara.

1. Abra Configurações > Personalização > Tela de fundo. Escolha uma imagem ou cor sólida
2. Ainda em Personalização > Cores, escolha o modo claro ou escuro
3. Ajuste o volume: clique no ícone de som na barra de tarefas e mova o controle
4. Ajuste o brilho: Configurações > Sistema > Tela > mova o controle de brilho

> **Dica:** Você pode clicar com o botão direito na área de trabalho para acessar opções rápidas de personalização.

### Missão 3 — Criar Pastas de Treinamento

Toda a sua organização no curso começa aqui.

1. Abra o Explorador de Arquivos (`Win + E`)
2. Navegue até **Documentos** no painel lateral esquerdo
3. Clique com o botão direito em uma área vazia > Novo > Pasta
4. Nomeie a pasta como `Treinamento PC`
5. Abra a pasta `Treinamento PC` e crie dentro dela três subpastas: `Documentos`, `Imagens`, `Projetos`

> **Dica:** O atalho `Ctrl + Shift + N` cria uma pasta nova direto, sem precisar do menu.

### Missão 4 — Capturar a Tela

Prints de tela são úteis para documentar o que você faz.

1. Pressione `Win + Shift + S`. A tela vai escurecer e aparecer uma barra no topo
2. Selecione a área que quer capturar arrastando o mouse
3. A imagem fica copiada. Abra o Paint (pesquise "Paint" no menu Iniciar) e cole com `Ctrl + V`
4. Salve a imagem na pasta `Treinamento PC > Imagens`

> **Dica:** `Win + Print Screen` captura a tela inteira e salva automaticamente em Imagens > Capturas de tela.

### Missão 5 — Conhecer Recursos de Acessibilidade

O Windows tem ferramentas que facilitam o uso do computador para todos.

1. Vá em Configurações > Acessibilidade
2. Explore a **Lupa** (Win + +): amplia partes da tela. Útil para textos pequenos
3. Conheça o **Narrador**: lê o conteúdo da tela em voz alta. Ative em Acessibilidade > Narrador
4. Ative as **Teclas de aderência** (Sticky Keys) em Acessibilidade > Teclado — permite pressionar atalhos de uma tecla por vez
5. Teste o **Alto contraste**: Acessibilidade > Temas de contraste — facilita a leitura

> **Dica:** Esses recursos não são apenas para pessoas com deficiência. A Lupa é ótima para apresentações e o ditado por voz (`Win + H`) acelera a digitação.

### Desafio da Semana

**Cenário:** Sua área de trabalho está cheia de arquivos e ícones espalhados.

**Resolver:**
1. Crie pastas na área de trabalho por categoria (Documentos, Imagens, Outros)
2. Mova cada arquivo para a pasta correspondente
3. Deixe a área de trabalho com no máximo 10 itens
4. Tire um print do resultado

### Projeto da Semana

Crie um documento no Google Docs chamado **"Meu Primeiro Computador"**.

Conteúdo:
- O que é um computador e para que serve
- O que você aprendeu nesta semana
- Cole o print da sua área de trabalho organizada

## Semana 2 — Arquivos

> Objetivo: criar, renomear, copiar, mover e encontrar arquivos no computador
> Tempo estimado: 2–3 horas

### Missão 1 — Criar Arquivos de Texto

1. Abra o **Bloco de Notas** (pesquise "Bloco de Notas" no menu Iniciar)
2. Escreva qualquer texto
3. Salve com `Ctrl + S` na pasta `Treinamento PC > Documentos`
4. Crie pelo menos 3 arquivos de texto diferentes

> **Dica:** Preste atenção na extensão do arquivo. Arquivos do Bloco de Notas são `.txt`.

### Missão 2 — Renomear Arquivos

1. Abra o Explorador de Arquivos e vá até a pasta onde criou os arquivos
2. Clique uma vez em um arquivo para selecioná-lo
3. Pressione `F2` para renomear
4. Dê nomes descritivos (ex: `anotacoes-semana2.txt` em vez de `Novo Documento.txt`)
5. Renomeie todos os seus arquivos com nomes claros

> **Dica:** Evite caracteres especiais nos nomes (/ \\ : * ? " < > |). Use hífens ou underlines no lugar de espaços.

### Missão 3 — Copiar e Mover Arquivos

1. Selecione um arquivo e copie com `Ctrl + C`
2. Navegue até outra pasta e cole com `Ctrl + V` — agora existem duas cópias
3. Para **mover** (sem deixar cópia), use `Ctrl + X` (recortar) e depois `Ctrl + V`
4. Pratique movendo arquivos entre as pastas `Documentos`, `Imagens` e `Projetos`

> **Dica:** Você pode selecionar vários arquivos de uma vez segurando `Ctrl` e clicando em cada um.

### Missão 4 — Encontrar Arquivos Perdidos

1. Pressione `Win + S` para abrir a busca do Windows
2. Digite parte do nome de um arquivo que você criou
3. Observe os resultados — o Windows mostra onde o arquivo está
4. No Explorador de Arquivos, use a barra de busca no canto superior direito para buscar dentro de uma pasta específica

### Desafio da Semana

**Cenário:** Você recebeu uma pasta com 10 arquivos bagunçados, sem nomes claros (Documento1.txt, foto.jpg, arquivo.pdf etc).

**Resolver:**
1. Crie uma pasta chamada `Organizado` dentro de `Treinamento PC`
2. Dentro dela, crie subpastas por tipo: `Textos`, `Imagens`, `PDFs`
3. Renomeie cada arquivo com um nome descritivo
4. Mova cada arquivo para a subpasta correta

### Projeto da Semana

Crie uma pasta chamada **"Projeto Escola"** dentro de `Treinamento PC > Projetos`.

Dentro dela:
- Crie 3 arquivos de texto com anotações sobre o que você aprendeu
- Organize tudo com nomes claros
- Tire um print da pasta organizada

## Semana 3 — Internet

> Objetivo: pesquisar no Google de forma eficaz, usar favoritos e baixar arquivos
> Tempo estimado: 2–3 horas

### Missão 1 — Pesquisar no Google

Saber pesquisar é uma das habilidades mais importantes.

1. Abra o navegador e vá ao Google
2. Pesquise cada um destes termos e observe os resultados:
   - `clima [sua cidade]`
   - `"como criar pasta no Windows 11"` (com aspas)
   - `como limpar PC -registry` (o menos exclui termos)
3. Compare: pesquise "computador lento" vs "Windows 11 lento ao iniciar" — qual resultado é mais útil?

> **Dica:** Quanto mais específica a pesquisa, melhor o resultado. Adicione o nome do programa ou versão do Windows.

### Missão 2 — Salvar Favoritos

Favoritos são atalhos para sites que você quer acessar de novo.

1. Abra um site útil (ex: Google Drive)
2. Pressione `Ctrl + D` para adicionar aos favoritos
3. Escolha uma pasta ou crie uma chamada "Curso"
4. Repita para pelo menos 5 sites úteis
5. Acesse seus favoritos pela barra de favoritos ou pelo menu do navegador

### Missão 3 — Usar Abas do Navegador

1. Abra uma nova aba com `Ctrl + T`
2. Abra 3 sites diferentes em abas separadas
3. Navegue entre elas clicando ou usando `Ctrl + Tab`
4. Feche uma aba com `Ctrl + W`
5. Reabra a aba fechada com `Ctrl + Shift + T`

### Missão 4 — Baixar Arquivos

1. Pesquise no Google "exemplo PDF download" e baixe um arquivo PDF
2. Pesquise uma imagem, clique com botão direito > "Salvar imagem como"
3. Encontre os arquivos baixados: abra o Explorador de Arquivos e vá até a pasta **Downloads**
4. Mova os arquivos baixados para as pastas corretas em `Treinamento PC`

> **Dica:** No navegador, pressione `Ctrl + J` para ver todos os downloads recentes.

> **Dica de Segurança:** Cuidado ao baixar arquivos! Verifique se o site tem o cadeado (HTTPS). Nunca baixe arquivos `.exe` de sites desconhecidos. Em sites de download, o botão verdadeiro geralmente é pequeno — os botões grandes e coloridos costumam ser propaganda.

### Desafio da Semana

**Cenário:** Você baixou um arquivo importante mas não sabe onde ele foi parar.

**Resolver:**
1. Abra o Explorador de Arquivos (`Win + E`)
2. Vá até a pasta Downloads
3. Ordene por data (clique no cabeçalho "Data de modificação") para achar os mais recentes
4. Se não encontrar, use a busca do Windows (`Win + S`) com parte do nome

### Projeto da Semana

Crie um documento no Google Docs chamado **"Top 5 Sites Úteis"**.

Conteúdo:
- Nome de cada site
- Link
- Para que serve
- Adicione todos os 5 aos seus favoritos organizados em uma pasta

## Semana 4 — Organização

> Objetivo: criar estruturas de pastas, ordenar arquivos e trabalhar com ZIP
> Tempo estimado: 2–3 horas

### Missão 1 — Criar Estrutura de Pastas para um Projeto

1. Pense em um projeto (pode ser fictício): "Trabalho de História"
2. Na pasta `Treinamento PC > Projetos`, crie a pasta do projeto
3. Dentro dela, crie subpastas: `Pesquisa`, `Imagens`, `Rascunhos`, `Final`
4. Crie alguns arquivos de teste em cada pasta

> **Dica:** Uma boa estrutura de pastas evita perder arquivos. Sempre separe por categoria.

### Missão 2 — Ordenar Arquivos

1. Abra qualquer pasta com vários arquivos
2. Clique no cabeçalho **Nome** para ordenar por nome (A-Z ou Z-A)
3. Clique em **Tamanho** para ver os maiores/menores
4. Clique em **Data de modificação** para ver os mais recentes
5. Clique com botão direito > Exibir e teste as opções: Ícones grandes, Lista, Detalhes

### Missão 3 — Criar Arquivo ZIP

ZIP é um formato que compacta arquivos, reduzindo o tamanho e juntando vários em um só.

1. Selecione uma pasta ou vários arquivos (segure `Ctrl` e clique em cada um)
2. Clique com botão direito > "Compactar para arquivo ZIP"
3. Renomeie o arquivo ZIP com um nome descritivo
4. Observe: o arquivo ZIP é menor que os arquivos originais somados

### Missão 4 — Extrair Arquivo ZIP

1. Encontre o arquivo ZIP que você criou
2. Clique com botão direito > "Extrair tudo"
3. Escolha onde extrair (pode ser na mesma pasta)
4. Clique em "Extrair" e confira se os arquivos estão lá dentro

> **Dica:** Você pode também abrir um ZIP com dois cliques para ver o conteúdo sem extrair.

### Desafio da Semana

**Cenário:** Alguém te enviou um arquivo ZIP com materiais de estudo. Você precisa organizar o conteúdo.

**Resolver:**
1. Extraia o arquivo ZIP
2. Analise o conteúdo e crie pastas por categoria
3. Renomeie arquivos com nomes claros
4. Mova cada arquivo para a pasta correta
5. Comprima a pasta organizada em um novo ZIP

### Projeto da Semana

Organize toda a sua pasta `Treinamento PC`:
- Verifique se todos os arquivos das semanas anteriores estão nas pastas certas
- Renomeie arquivos que ainda tenham nomes genéricos
- Crie um arquivo de texto chamado `indice.txt` listando tudo que tem dentro
- Comprima toda a pasta `Treinamento PC` em um arquivo ZIP
