# Academia do PC

## Programa de 6 Meses para Dominar o Computador

**Objetivo:** Usar o computador melhor que 90% das pessoas.

Este treinamento usa apenas ferramentas gratuitas do **Windows 11**, navegador e **Google Docs/Planilhas**.

---

## Como Usar Este Guia

**Ritmo sugerido:** 1 semana por bloco. Cada semana tem missões, um desafio prático e um projeto. Faça no seu ritmo — o importante é entender, não correr.

**Filosofia: tente > pesquise > pergunte.** Antes de pedir ajuda:
1. Tente resolver sozinho
2. Pesquise no Google ou YouTube
3. Se ainda não conseguiu, pergunte (a alguém ou a uma IA)

**Como marcar progresso:** Cada semana tem um checklist no final. Marque os itens conforme completar. Se usar o GitHub, os checkboxes funcionam direto no markdown.

---

## Aprenda a Se Virar

Saber pesquisar é a habilidade mais importante da informática. Quem sabe pesquisar resolve qualquer problema.

### Como pesquisar no Google

- **Use termos específicos:** em vez de "não funciona", pesquise "Windows 11 Wi-Fi desconectando sozinho"
- **Use aspas para termos exatos:** `"gerenciador de tarefas"` encontra essa frase exata
- **Adicione "como" no início:** "como compactar pasta em ZIP Windows 11"
- **Filtre por data:** clique em Ferramentas > Qualquer data > Último ano (para problemas recentes)

### Como usar YouTube para aprender

- Pesquise o que quer fazer + "tutorial" (ex: "Google Docs tutorial português")
- Use a velocidade 1.25x ou 1.5x para economizar tempo
- Pause e faça junto com o vídeo

### Como usar IA para tirar dúvidas

- Use o ChatGPT, Claude ou Gemini (todos gratuitos)
- Seja específico: em vez de "me ajuda com o PC", diga "como faço para desinstalar um programa no Windows 11"
- Peça para explicar de forma simples se a resposta for técnica demais

## Glossário

Termos técnicos que aparecem no curso, explicados de forma simples.

| Termo | O que é |
|---|---|
| **CPU (Processador)** | O "cérebro" do computador. Faz os cálculos e executa os programas |
| **RAM (Memória)** | Memória temporária. Quanto mais RAM, mais programas abertos ao mesmo tempo |
| **SSD / HD** | Onde ficam seus arquivos guardados. SSD é mais rápido que HD |
| **GPU (Placa de vídeo)** | Processa imagens e vídeos. Importante para jogos e edição |
| **Sistema Operacional** | O programa principal (Windows 11). Controla tudo no computador |
| **Navegador** | Programa para acessar a internet (Chrome, Edge, Firefox) |
| **Extensão de arquivo** | As letras depois do ponto no nome do arquivo (.pdf, .docx, .jpg) |
| **ZIP** | Formato de arquivo compactado. Junta vários arquivos em um só, menor |
| **Firewall** | Proteção que controla o que entra e sai do computador pela rede |
| **Phishing** | Golpe que imita sites ou emails reais para roubar seus dados |
| **HTTPS** | O cadeado no site. Significa que a conexão é segura |
| **IP** | Endereço do computador na rede, como um CEP digital |
| **DNS** | Traduz nomes de sites (google.com) para endereços IP |
| **Driver** | Programa que faz o Windows conversar com o hardware (impressora, placa de vídeo) |
| **Backup** | Cópia de segurança dos seus arquivos |
| **Nuvem** | Armazenamento na internet (Google Drive, OneDrive) |
| **Cache** | Dados temporários guardados para carregar coisas mais rápido |
| **Bluetooth** | Tecnologia sem fio para conectar dispositivos próximos (fones, teclados, mouses) |
| **USB** | Porta e conector universal para ligar dispositivos ao computador (pendrives, cabos, carregadores) |
| **HDMI** | Cabo e porta para transmitir imagem e som para monitores e TVs |
| **2FA / Verificação em duas etapas** | Camada extra de segurança que pede uma confirmação além da senha (código por SMS ou app) |
| **Malware** | Programa malicioso — qualquer software criado para causar dano (vírus, spyware, ransomware) |
| **Ransomware** | Tipo de malware que bloqueia seus arquivos e pede dinheiro para devolvê-los |
| **Terminal / Prompt de Comando / CMD** | Tela onde você digita comandos em texto para controlar o computador |
| **PowerShell** | Terminal mais moderno do Windows, com comandos mais poderosos que o CMD |
| **Script** | Arquivo de texto com uma sequência de comandos que o computador executa automaticamente |

## Quando Algo Der Errado

Guia rápido para resolver problemas comuns.

### O computador está lento
1. Abra o Gerenciador de Tarefas (`Ctrl + Shift + Esc`)
2. Veja qual programa está usando mais CPU ou Memória
3. Se algum programa estiver usando muito e não for importante, clique nele e em "Finalizar Tarefa"
4. Se continua lento, reinicie o computador

### Um programa travou
1. Abra o Gerenciador de Tarefas (`Ctrl + Shift + Esc`)
2. Encontre o programa travado (pode aparecer "Não Respondendo")
3. Clique nele e em "Finalizar Tarefa"
4. Abra o programa novamente

### A internet não funciona
1. Verifique se o Wi-Fi está ligado (ícone na barra de tarefas)
2. Desconecte e reconecte ao Wi-Fi
3. Reinicie o roteador (desligue, espere 30 segundos, ligue)
4. Se nada funcionar, abra o Prompt de Comando e digite `ping google.com` para testar a conexão

### Não consigo encontrar um arquivo
1. Use `Win + S` e digite parte do nome do arquivo
2. Abra o Explorador de Arquivos (`Win + E`) e use a barra de busca
3. Verifique as pastas Downloads, Documentos e Área de Trabalho
4. Verifique a Lixeira — pode ter sido deletado sem querer

### O computador não liga
1. Verifique se está conectado na tomada
2. Se for notebook, conecte o carregador e espere alguns minutos
3. Segure o botão de ligar por 10 segundos, solte, e pressione novamente
4. Se não funcionar, procure assistência técnica

### Apareceu uma mensagem estranha / pop-up suspeito
1. Não clique em nada na mensagem
2. Feche a aba ou janela (use `Alt + F4` se necessário)
3. Execute uma verificação com o Windows Defender
4. Se a mensagem voltar sempre, pode ser um programa indesejado — veja como desinstalar no [Mês 4](mes-4-diagnostico.md)
