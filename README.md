# 🧠 Academia do PC
## Programa de 6 Meses para Dominar o Computador

Objetivo final:  
👉 **Usar o computador melhor que 90% das pessoas.**

Este treinamento usa apenas ferramentas gratuitas do **Windows 11**, navegador e **Google Docs**.

---

# 🎮 Sistema de Jogo

## XP (Experiência)

- Missão simples → **10 XP**
- Missão média → **20 XP**
- Desafio prático → **30 XP**
- Projeto semanal → **50 XP**
